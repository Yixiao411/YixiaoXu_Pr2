# Pagina Web

## Pagina Inicial:
![imatge pagina principal](/markdown/index.png)

La pagina principal esta dividido en diversos secciones, sobre la distribucion de cartar he intentado seguir el diseño original de ejemplo intentando mostrar de manera con flex y tambien grid. Luego para el imagen de la portada he probado de hacer un corte sobre la imagen para adaptar el diseño.

Sobre la organizacion de html he decidido de porner el contenido mas inportante(parte superior) como main, y lo resto en seccion, tanto en pagina principal o bien en las subpaginas.

## Pagina Musica:
![imatge pagina lanzamientos](/markdown/release.png)

La pagina de lanzamientos he hecho un diseño como de playlist que muestra una lista de canciones que estan populares y en el main la cancion mas importante con una pequeña informacion. Y tambien una muiestra de audio para reproducir.

## Pagina detalle:
![imatge pagina detalle](/markdown/detall.png)

En el pagina de detalle se mostrara mas informacion detallada sobre la cancion y un imagen de ello, diseño simple, y luego por parte inferior un boton para volver al listado.

## Pagina Alta
![imatge alta](/markdown/alta.png)

La utima pagina para darse de alta estan divididas por dos formularios distintos, una para hacer registro como usuario, y por debajo el formulario para hacer el contacto para informar cualquier cosa, introduciendo con un texto de explicacion.
